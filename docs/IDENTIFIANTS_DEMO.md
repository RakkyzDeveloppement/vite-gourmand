# Identifiants Demo

## Admin
- Email: `admin@vite-gourmand.fr`
- Mot de passe: `password`

## Employe
- A creer depuis l'admin avant la soutenance.
- Exemple:
  - Email: `employe1@vite-gourmand.fr`
  - Mot de passe: `Employe123!`

## Utilisateur
- A creer via formulaire d'inscription.
- Exemple:
  - Email: `client1@exemple.fr`
  - Mot de passe: `Client123!`

## Donnees de test conseillees
- Au moins 2 commandes:
  - 1 commande statut `recue` (pour montrer modification/cancel)
  - 1 commande statut `terminee` (pour montrer avis)
- Au moins 1 synchronisation NoSQL effectuee dans admin.
